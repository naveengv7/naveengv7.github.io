Can Machine Learning Algorithms Classify Inner Speech from EEG Brain Signals?

Authors: Nicholas R Merola, Naveenkumar G Venkataswamy, Masudul H Imtiaz

Conference: 2023 IEEE World AI IoT Congress (AIIoT), June 7-10, 2023

DOI: 10.1109/AIIoT58121.2023.10174580

Summary

Objective: This study explores classifying inner speech using EEG signals and machine learning models for brain-computer interfaces (BCIs).

Methodology:

  • EEG signals from 8 subjects thinking about four directional words.
  • Data recorded using a BioSemi ActiveTwo system (128 EEG channels).
  • Preprocessing: Filtering, re-referencing, feature extraction using ANOVA ranking.
  • Models Used: Random Forest, SVM, KNN.
  • Best accuracy: SVM with a quadratic kernel (35.3%).

Key Findings: While classification accuracy is low, results confirm EEG-based inner speech recognition feasibility.

Conclusion: Machine learning can classify inner speech, paving the way for thought-controlled assistive technologies.
