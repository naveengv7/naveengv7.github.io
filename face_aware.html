An Open-Source Face-Aware Capture System

Authors: Md Abdul Baset Sarker, SM Safayet Hossain, Naveenkumar G Venkataswamy, Stephanie Schuckers, Masudul H Imtiaz

Conference: Electronics 2024

DOI: https://doi.org/10.3390/electronics13071178

Objective

This study presents an open-source face-aware capture system designed to improve the quality of facial images used in biometric authentication, especially for passport photo acquisition and verification. The system leverages computer vision algorithms and real-time face quality assessment to ensure high-quality image capture.

System Overview

  • Hardware: Includes an NVIDIA Jetson processor, a 16 MP autofocus RGB camera, a custom enclosure, and a touchscreen LCD.
  • Software: Uses open-source algorithms to evaluate face quality based on ISO standards.
  • Security: Implements AES encryption to ensure secure image storage and prevent data tampering.

Methodology

  • The system detects a user's face and evaluates its quality based on pose, lighting, and clarity.
  • Facial images are assessed for compliance with ISO standards using geometric and photographic tests.
  • Users can select the best-quality image from a set of captured photos.
  • Once an image is selected, it is encrypted using AES for secure storage.

Results

  • The system achieved an accuracy of 98.98% in capturing high-quality facial images.
  • Testing on the U.S. passport verification tool confirmed its effectiveness.
  • Real-time processing allows for **fast and efficient** passport photo capture.

Conclusion

The proposed face-aware capture system enhances biometric authentication by ensuring the capture of **high-quality, ISO-compliant images**. Its open-source nature makes it highly adaptable for various applications, including **passport verification, security, and identity authentication**. Future improvements will focus on **deep learning-based enhancements, automated camera positioning, and expanded use cases**.
