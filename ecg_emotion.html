Predicting Emotional States from Wearable ECG

Authors: Timothy C Sweeney-Fanelli, Ernesto Sola-Thomas, Naveenkumar G Venkataswamy, Justin C Au-Yeung, Masudul H Imtiaz

Conference: 2024 IEEE 15th Annual Ubiquitous Computing, Electronics & Mobile Communication Conference (UEMCON)

DOI: 10.1109/UEMCON62879.2024.10754745

Objective

This study explores a machine learning-based approach to predicting **emotional responses** using **ECG signals**. By analyzing ECG data from the DREAMER dataset, the goal is to classify emotions based on valence, arousal, and dominance metrics, contributing to **mental health monitoring** and **empathetic technology**.

Methodology

  • Dataset: ECG signals from 23 individuals exposed to various multimedia stimuli.
  • Preprocessing: Noise reduction using **heartpy**, baseline drift removal, and **bandpass filtering**.
  • Feature Extraction: A set of **13 time-domain features** (e.g., bpm, sdnn, rmssd, hrmad) extracted from ECG signals.
  • Machine Learning Models Used:
    • Random Forest (RF)
    • Support Vector Machine (SVM)
    • Logistic Regression
  • Model Optimization: Applied **Principal Component Analysis (PCA)** and hyperparameter tuning.

Results

  • Random Forest: Achieved the highest accuracy of **82%** for emotion classification.
  • SVM: Performed well with **40% accuracy**, but struggled with generalization.
  • Logistic Regression: Showed a peak accuracy of **38%**.
  • PCA improved classification performance by simplifying features while retaining **95% variance**.

Conclusion

This research confirms that **ECG-based emotion classification is feasible** using machine learning. The **Random Forest model performed best**, suggesting potential for **real-world emotion detection applications**. Future work will involve refining deep learning techniques, expanding datasets, and improving feature selection for **better accuracy in predicting emotional responses**.
