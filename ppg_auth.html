User Authentication and Vital Signs Extraction from Low-Frame-Rate and Monochrome No-contact Fingerprint Captures

Authors: Olaoluwayimika Olugbenle, Logan Drake, Naveenkumar G Venkataswamy, Arfina Rahman, Yemi Afolayanka, Masudul Imtiaz, Mahesh K Banavar

Conference: arXiv preprint arXiv:2412.07082

DOI: 10.48550/arXiv.2412.07082

Objective

This study explores the use of low-frame-rate, monochrome (blue light) fingerprint captures to extract **vital signs** and authenticate users. By leveraging photoplethysmography (PPG), the system detects heart rate and other biometric signals, enhancing **liveness detection** for security applications.

Methodology

  • Fingerprint Sensor: A **no-contact fingerprint scanner** captures fingertip images at 3000 PPI resolution and **14 frames per second**.
  • PPG Signal Extraction: Light intensity variations in the fingertip images are analyzed to measure heart rate and oxygen saturation.
  • Vital Signs Calculation: The system applies a **Chebyshev low-pass filter** and moving average filters to enhance accuracy.
  • User Authentication: Two methods were tested:
    • Human ID Approach: Statistical pattern analysis of PPG signals.
    • Deep Learning-Based Approach: A CNN-LSTM model trained on time-series PPG data.

Results

  • Vital Signs Estimation: The **mean error rate in heart rate detection was reduced from 22.9% to 11.6%** with improved filtering.
  • User Authentication (Human ID Method): Successfully identified users in most cases but had some false negatives.
  • Deep Learning Approach: Model accuracy fluctuated due to class imbalance, highlighting the need for further optimization.

Conclusion

This research demonstrates that **low-frame-rate, monochrome fingerprint sensors can be used for both liveness detection and biometric authentication**. The findings suggest that PPG signals obtained from fingerprint images can add a new layer of security against spoofing. Future work will focus on **expanding datasets, refining deep learning models, and improving accuracy** for real-world applications.
